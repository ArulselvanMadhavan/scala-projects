## Installation

ScalaWithCats is available on scala 2.11, 2.12, and scalajs.

Add the following to your build.sbt
```scala
libraryDependencies ++= Seq(

  "com.arulselvan" %% "ScalaWithCats-core" % "0.0.1")
```
